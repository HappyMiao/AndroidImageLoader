#### AndroidImageLoader

* 采用glide框架、glide-transformations框架进行封装，进行图片加载、图片处理（切圆角、灰度处理等）、图片缓存。

* 项目开源地址 [glide](https://github.com/bumptech/glide)、[glide-transformations](https://github.com/wasabeef/glide-transformations)
